# Frontend - Sistema de Gestión de Hostal

Frontend moderno desarrollado con React + TypeScript + Vite.

## Tecnologías

- React 18
- TypeScript
- Vite
- React Router
- TanStack Query
- Axios
- Tailwind CSS
- Zustand
- Lucide React (iconos)

## Instalación

```bash
npm install
```

## Configuración

Crear archivo `.env`:
```
VITE_API_BASE_URL=http://localhost:8000
```

## Desarrollo

```bash
npm run dev
```

Acceder a: http://localhost:3000

## Build

```bash
npm run build
npm run preview
```
